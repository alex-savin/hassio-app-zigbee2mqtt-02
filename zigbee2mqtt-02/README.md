# Home Assistant Add-on: Zigbee2MQTT Instance 02

Allows you to use your Zigbee devices **without** the vendors bridge or gateway.

It bridges events and allows you to control your Zigbee devices via MQTT. In this way you can integrate your Zigbee devices with whatever smart home infrastructure you are using.

This is **Instance 02** — a separate copy of the [Zigbee2MQTT add-on](https://github.com/zigbee2mqtt/hassio-zigbee2mqtt) with a unique slug (`zigbee2mqtt-02`) so it can run alongside the stock add-on and other instances, each managing its own Zigbee coordinator and network.

See the Documentation tab for configuration details and multi-instance setup instructions.
